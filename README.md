# 🎮 Number Series Game Hub (Java Mini Project)

An interactive Java-based console game designed to improve logical thinking and aptitude skills through number series puzzles.

---

## 🚀 Features

- 🧠 Multiple Game Modes:
  - Missing Number Game
  - Next Number Game
  - Pattern Guess Game
  - Rapid Fire Quiz

- 🎯 Difficulty Levels:
  - Easy
  - Medium (Unlockable)
  - Hard (Unlockable)

- ⏱️ Real-Time Countdown Timer (15 seconds per question)

- 🔓 Level Unlock System

- 💡 Smart Hints for each question

- 🔀 Random Pattern Generation:
  - Arithmetic Progression (AP)
  - Geometric Progression (GP)
  - Fibonacci Series
  - Alternating Patterns
  - Mixed Logic Patterns

---

## 🛠️ Technologies Used

- Java
- OOP Concepts
- Multithreading (Timer implementation)
- Randomization

---
